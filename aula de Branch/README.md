# Bem vindo a criação de uma nova branch:

Os comandos para criar uma nova branch são esses. Execute-os na sequência:

1. `git branch`
2. `git checkout <nome_da_branch>`
3. `git status` 
4. `git branch` - Verifica se a branch criada foi definida como default
5. `git add .` - adiciona todos os arquivos para o commit
8. `git commit -m "mensagem desejada` - cria o commit para ser enviado para a Branch 
9. `git push origin <nome_da_branch>` - envia as alterações para a branch desejada
